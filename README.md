This repository contains multiple data science and machine learning projects built using a global restaurant dataset. The goal is to explore restaurant trends, predict ratings, and build intelligent recommendation systems.

## Dataset

The dataset contains restaurant information from multiple cities across the world, including attributes such as:

- Cuisine types
- Restaurant location
- Price range
- Customer votes
- Online delivery availability

This dataset is used across multiple projects in this repository to explore different data science problems.

## Projects included:

1. [Restaurant Rating Prediction ✅](rating_prediction/)
2. Restaurant Rating Classification
3. Restaurant Recommendation System
4. Restaurant Success Prediction

## How to Run the Project

1. Clone the repository

    ```bash
    git clone https://github.com/RSurfer90/restaurant-data-science-projects.git
    ```

2. Navigate to the project folder

    ```bash
    cd restaurant-data-science-projects
    ```

3. Install required libraries

    ```bash
    pip install -r requirements.txt
    ```

4. Open the notebook

    ```bash
    jupyter notebook
    ```
